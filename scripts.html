🖥️ Server Script 📏 1118 lines 🎯 FFD System

Core server script for the full destruction sequence. Builds all interactive props (button, keypad, scanner, indicator lights) at runtime. Runs a state machine through the interaction flow. Keycard swipe, countdown phases, facility-wide light change, blast door and elevator lockdown, music sequencing. Then resets and respawns all players.
