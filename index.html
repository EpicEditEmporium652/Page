CODE

Scripts

Main.lua Server script for the full destruction sequence. Builds all interactive props (button, keypad, scanner, indicator lights) at runtime. Runs a state machine through the interaction flow. Keycard swipe, countdown phases, facility-wide light change, blast door and elevator lockdown, music sequencing. Then resets and respawns all players.
Shake.lua Nuclear explosion client script. Reads audio amplitude each frame and offsets the camera in real time, so the shake follows the audio waveform. Also applies physics to a hanging lamp and triggers a real lighting exposure flash. Not a screen overlay.
ProximityPrompt.lua Full replacement for Roblox's default proximity prompt. Custom UI with icon, action label, hold-to-activate bar, hover highlight, and sound on show. Detects touch vs keyboard automatically. Assets preload to avoid pop-in.
Admin.lua Admin system. Prefix-based chat commands with me / all / others / partial name targeting. Bans and ranks persist via DataStore across server restarts. Supports offline ranking. You can promote or ban a player who isn't currently in the game.
Phase1.lua Phase 1 client. On server signal, lights the phase indicator and triggers the screen display.
Phase2.lua Phase 2 client. Receives a timing value from the server and runs the full progress bar animation locally, 0–100% at whatever speed is specified.
Phase3.lua Phase 3 client. Single server fire triggers the activation sound, notification, and a live countdown that ticks every second until the sequence is ready.
Serverviewer_client.lua Client side of a real-time server state visualiser. Renders a top-down spectator view with live position updates streamed each frame and part path caching to keep it performant. (code only, no live demo)
Serverviewer_server.lua Server side. Collects and fires position data for every relevant part each frame. Disconnects cleanly when the viewer is toggled off. (code only, no live demo)
PLUG AND PLAY
Notification.lua Plug-and-play notification module. Call module.sg("text", duration) from any LocalScript. Handles fade-in, fade-out, and cleanup automatically
